# CLI Task Tracker
CLI Task Tracker is a command-line tool for you to manage your tasks. This tool allows you to add, update, delete, and list your tasks as well as managing their stasuses.

## Project link
```
https://roadmap.sh/projects/task-tracker

```

## How to install?
### 1. Clone the Repository
```bash
git clone https://github.com/Araayaaa/task-tracker-cli.git
```

### 2. Make the script executable (Optional, for Linux/macOS)
```bash
chmod +x index.js
```

### 3. Link the command using npm
```bash
npm link
```
